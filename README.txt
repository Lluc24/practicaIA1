Hay un fichero ejecutable .jar con path relativo out/artifacts/practicaIA1_jar/practicaIA.jar
Para ejecutar el .jar hay que
	1. Situarse en el mismo directorio donde hay el README.txt
	2. Ejecutar java -jar out/artifacts/practicaIA1_jar/practicaIA.jar

El programa es interactivo mediante terminal. Hace preguntas y mediante las respuesta se puede jugar con todas las funcionalidades.
Los algoritmos que se pueden usar son Simulated Annealing y Hill Climbing.
Las funciones heuristicas implementadas son:
	1. Coste de transporte y almacenamiento
	2. Coste de transporte y almacenamiento mas felicidad. Ponderados por los parametros A y B.

El codigo se ha desarrollado desde Jetbrains IntelliJ Idea. Recomendamos abrir el proyecto con este IDE.

Muchas gracias por leer!
